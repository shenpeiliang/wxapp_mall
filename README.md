### 功能简介

1、商城首页、商品列表、商品详情、商品分类、会员中心、购物车、订单等商城基本功能基本都可以用；

2、使用微信授权登录，微信收货地址；

3、该代码只包含微信小程序代码，没有上传后台服务端代码；
